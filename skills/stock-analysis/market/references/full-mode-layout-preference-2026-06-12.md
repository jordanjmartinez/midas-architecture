# Full-Mode Layout Preference — Profile First, Trend Section

Date: 2026-06-12

## Trigger

Use this note when maintaining or extending `!market full` / `!market expanded` output.

## User preference captured

For normal successful full/expanded market snapshots:

1. Keep `Source: [provider]` directly below `As of:`.
2. Put `## Profile` immediately after the header/source block, before `## Price Action`.
3. Use a separate `## Trend` section because `## Liquidity` is already its own section.
4. Keep profile fields rather than removing them; the issue was placement, not usefulness.
5. Do not change provider selection, fallback behavior, supplement calls, raw/debug output, watchlists, artifacts, scoring, or research behavior for layout-only patches.
